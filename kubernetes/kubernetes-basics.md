# Kubernetes 基礎

### 什麼是Kubernetes

Kubernetes是一套工具，用以管理在不同主機上運行的不同容器\(container\)，當我們需要運行多個Docker Image來產生多個container時，Kubernetes能夠給我們很大的幫助。
